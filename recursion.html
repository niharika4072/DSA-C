Recursion

Recursion is a programming technique in which a function calls itself directly or indirectly to solve a problem by breaking it down into smaller, simpler sub-problems. It’s widely used in algorithms and data structures for tasks that can be defined in terms of similar sub-tasks, like traversing tree structures, performing mathematical computations, or sorting data.


Key Characterstics of Recursion

  • Base Case: A condition that stops the recursion by returning a direct answer, preventing infinite calls. Without a base case, recursion would continue indefinitely.
  • Recursive Case: The part where the function calls itself with a modified argument, gradually approaching the base case.

Types of Recursion

  • Direct Recursion: When a function calls itself directly.
  • Indirect Recursion: When a function calls another function that eventually calls the original function.
  • Tail Recursion: When the recursive call is the last operation in the function, making it easier for some compilers to optimize by reusing stack frames (known as tail call optimization).
  • Head Recursion: When the recursive call occurs before any other operations in the function.

Recursive VS Iterative Approach

While recursion can simplify code, it’s often possible to convert a recursive solution into an iterative one. Iteration generally consumes less memory and can be faster, as it doesn’t involve the overhead of multiple function calls, but recursion can make the code more intuitive and easier to understand for problems with a recursive nature.

Applications of Recursion

  • Mathematics and Computations: Calculations like powers, greatest common divisor, factorials, and Fibonacci numbers.
  • Data Structures: Traversing trees and graphs, and manipulating nested structures.
  • Divide and Conquer Algorithms: Sorting algorithms like merge sort and quicksort use recursion to break down and sort sub-arrays.
  • Dynamic Programming: Optimized recursive solutions are frequently used in dynamic programming problems where sub-solutions are cached to avoid redundant calculations.

1.CALCULATE FACTORIAL OF A NUMBER USING RECURSION

#include<stdio.h>

int factorial(int n)
{
    if(n==0)
    {
        return 1;
    }
    else
    {
        return (n*factorial(n-1));
    }
}

int main()
{
    int n;
    printf("Enter the number for finding factorial :");
    scanf("%d",&n);
    printf("FACTORIAL OF %d = %d",n,factorial(n));
}
        
                
OUTPUT

Enter the number for finding factorial :4
FACTORIAL OF 4 = 24                  
        
        

2.CALCULATE FIBONACCI OF A NUMBER USING RECURSION.

#include <stdio.h>

int fibonacci(int n) 
{
    if (n <= 1) 
    {
        return n;
    }
    return fibonacci(n - 1) + fibonacci(n - 2);
}

int main() 
{
    int n;
    printf("Enter a non-negative integer: ");
    scanf("%d", &n);
    
    if (n < 0) 
    {
        printf("Fibonacci is undefined for negative numbers.\n");
    } 
    else 
    {
        printf("Fibonacci sequence up to %d:\n", n);
        for (int i = 0; i <= n; i++) 
        {
            printf("%d ", fibonacci(i));
        }
        printf("\n");
    }
    
    return 0;
}
                

OUTPUT

Enter a non-negative integer: 5
Fibonacci sequence up to 5:
0 1 1 2 3 5  

        

3.CALCULATE THE SUM OF NATURAL NUMBERS TILL 10 USING RECURSION.

#include <stdio.h>

int sumNatural(int n) 
{
    if (n <= 0) 
    {
        return 0;
    }
    return n + sumNatural(n - 1);
}

int main() 
{
    int result = sumNatural(10);
    printf("The sum of natural numbers up to 10 is: %d\n", result);
    return 0;
}



OUTPUT

The sum of natural numbers up to 10 is: 55

5.FIND THE LARGEST ELEMENT IN THE INTEGER ARRAY USING RECURSION.

#include <stdio.h>

int findLargest(int arr[], int size) 
{
    if (size == 1) 
    {
        return arr[0];
    }
    
    int largest = findLargest(arr, size - 1);
    return arr[size - 1] > largest ? arr[size - 1] : largest;
}

int main() 
{
    int arr[] = {3, 5, 7, 2, 8, 1, 4};
    int size = sizeof(arr) / sizeof(arr[0]);
    
    int largest = findLargest(arr, size);
    printf("The largest element in the array is: %d\n", largest);
    
    return 0;
}


OUTPUT

The largest element in the array is: 8

8.FIND THE ARITHMETIC MEAN OF AN INTEGER ARRAY USING RECURSION.

#include <stdio.h>

float calculateSum(int arr[], int size) 
{
    if (size == 0) 
    {
        return 0;
    }
    return arr[size - 1] + calculateSum(arr, size - 1);
}

float calculateMean(int arr[], int size) 
{
    if (size == 0) 
    {
        return 0; // To avoid division by zero
    }
    float sum = calculateSum(arr, size);
    return sum / size;
}

int main() 
{
    int size, i;
    printf("Enter the number of elements in the array: ");
    scanf("%d", &size);
    int arr[size];
    printf("Enter %d elements:\n", size);
    for (i = 0; i < size; i++) 
    {
        scanf("%d", &arr[i]);
    }

    float mean = calculateMean(arr, size);
    printf("The arithmetic mean of the array is: %.2f\n", mean);
    
    return 0;
}


OUTPUT

Enter the number of elements in the array: 5
Enter 5 elements:
2
4
5
52
2222
The arithmetic mean of the array is: 457.00
